# Tài liệu lưu trữ và phụ trợ

Thư mục này chứa các tài liệu không thuộc bộ báo cáo chính, nhưng vẫn được giữ lại để tham chiếu lịch sử, kiểm thử, debug hoặc quá trình triển khai.

## Nhóm tài liệu trong archive

- `plans/`
  - Các kế hoạch tính năng và backlog triển khai.
- `UI-REVIEW.md`
  - Báo cáo review giao diện và khả dụng.
- `backend-security-review-2026-05-07.md`
  - Báo cáo review bảo mật backend.
- `security-scan-2026-05-07.md`
  - Kết quả quét bảo mật và phụ thuộc.
- `EVALUATION_EVIDENCE.md`
  - Minh chứng đánh giá và kết quả kiểm thử mở rộng.
- `OCR_EXCEL_FEATURE.md`
  - Mô tả tính năng OCR/Excel chuyên biệt.
- `OCR_EXCEL_IMPLEMENTATION.md`
  - Chi tiết triển khai OCR/Excel chuyên biệt.
- `railway-redis-debug.md`
  - Ghi chú debug Redis/Railway.
- `rq-worker-fix.md`
  - Ghi chú sửa lỗi worker RQ.

## Bộ báo cáo chính

Các tài liệu phục vụ báo cáo/chấm chính nằm ở thư mục `docs/` gốc, bắt đầu từ:

- `docs/ARCHITECTURE_DOCUMENT_SET.md`
- `docs/SYSTEM_DESIGN_OVERVIEW.md`
- `docs/ARCHITECTURE_DIAGRAMS.md`
- `docs/AI_LOGS.md`
